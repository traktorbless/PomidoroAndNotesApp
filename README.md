# Pomodoro time manager and notes

# RUS
С помощью этого приложения вы можете ставить задачи и выполнять их используя метод "Pomodoro", а также делать заметки.

Верстка осуществлялась с помощью **SwiftUI**.

Для хранения данных используется **CoreData**.

Также в приложении используются локальные уведомления.

# ENG

With this app You can set tasks and complete them using method "Pomodoro", also you can make notes.

Layout has been done with **SwiftUI**

Data has been storaged with **CoreData**

Also local notifications is used in app
